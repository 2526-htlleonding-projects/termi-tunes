# User Stories

## CLI

The CLI shoud focus on being easy and fast to use, much should happen automatically in the background so if a user knows
the commnds well and ha a good organisation, he gets reawred.

### Examples

- I want to play the playlist "mode." in smart shuffle mode
```
play -ss "mode."
```

- I want to activate shuffle mode
```
play -s
```

- I want to add the song which is currently playing to the playlist i'm listening to at the moment
```
add
```

- I want to see the lyrics of the song to look up a specific part
```
lyr -a
```

- I want to play my favoutires
```
play #0
```

- I want to look for a song called "victory lap" and play the first result
```
search -f "victory lap"
```

- I want to play music from my speaker setup
````
ldevice

1 - speaker setup
2 - tv
3 - phone
4 - computer

 *press '1'
````

- I want to skip the current song
```
skip
```

## TUI

- I want a graphical interface where i can select my playlists and songs only using my keyboard.

- I want to see how long the song is and how long I have left.

- I want to see the artist of the current song and have the option to see more information about them.
